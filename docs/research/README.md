# Research Documentation

This folder contains customer research, domain research, market evidence, experiments, findings, assumptions, and open questions that inform product and strategy.

Research should distinguish evidence from interpretation and should identify which claims or decisions it may support. Design documents produced from research must state their dependencies on [`docs/canon`](../canon/README.md).
