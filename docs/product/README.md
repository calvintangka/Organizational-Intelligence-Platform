# Product Documentation

This folder contains product definitions derived from the foundational documents, including user problems, product requirements, workflows, experience specifications, feature decisions, and product policies.

Documents here should state which files in [`docs/canon`](../canon/README.md) they derive from and should distinguish durable product requirements from temporary implementation or roadmap choices.
