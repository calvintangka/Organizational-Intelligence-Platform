# Strategy Documentation

This folder contains company, product, market, platform, and expansion strategy derived from the company's foundational commitments.

Strategy documents should make their assumptions, trade-offs, and time horizon explicit and should state which files in [`docs/canon`](../canon/README.md) they derive from.
